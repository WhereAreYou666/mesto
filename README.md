# Проект: [Место](https://whereareyou666.github.io/mesto/)

## Описание проекта:
Одностраничный сайт для размещения фотографий

## Функционал
  * Адаптивная верстка;
  * Трансформация элементов;
  * Редактирование профиля;
  * Добавление карточки;
  * Лайк карточки;
  * Удаление карточки;
  * Валидация форм;
  * Закрытие кликом на оверлей;
  * Закрытие нажатием на Esc;

## Стек технологий:
  * HTML;
  * CSS;
  * JS;
  * DOM;
  * Figma;
  * Git;
  * GitHub Pages;
  * Bem;
  * Grid layout;
  * Flexbox;
